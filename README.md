# Server en Express.js

Este proyecto es un servidor básico desarrollado con Express.js. Proporciona endpoints para gestionar una lista de productos almacenados en un archivo JSON.

## Características

- **Gestión de productos:** Crear, leer, actualizar y eliminar productos.
- **Rutas:** Implementación de rutas para interactuar con los datos de productos.
- **Persistencia:** Los datos se almacenan en el archivo `products.json`.
